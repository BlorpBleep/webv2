# CicadaVPN Linux Installation Guide

## Step 1: Download the CicadaVPN App

1. Open your browser and go to the **CicadaVPN Downloads** page at [cicadavpn.com/downloads](https://www.cicadavpn.com/downloads).
2. Scroll down to the **Linux** section and download the package appropriate for your distribution (e.g., **Debian/Ubuntu** or **Fedora**).
3. Once the download is complete, open a terminal and navigate to the directory where the package is located.

## Step 2: Install the CicadaVPN App

### For Debian/Ubuntu:
1. Use the following command to install the package:

   ```bash
   sudo dpkg -i cicadavpn_*.deb
   ```

2. If there are dependency issues, run:

   ```bash
   sudo apt-get install -f
   ```

### For Fedora:
1. Use the following command to install the package:

   ```bash
   sudo dnf install cicadavpn_*.rpm
   ```

## Step 3: Open the CicadaVPN App

1. Once installed, launch the CicadaVPN app by typing the following command in your terminal:

   ```bash
   cicadavpn
   ```

2. You will be prompted to either **Log In** or **Sign Up**.
   - If you purchased your subscription directly from **cicadavpn.com**, your account is already activated, and you’re good to go. Simply log in with your email and password.
   - If you purchased a **physical activation code** from a store, follow the steps in **Step 4** to redeem your activation code.

## Step 4: Activate Your VPN Service (For Physical Activation Codes)

- **If you purchased a physical activation code from a store**:
  1. In the CicadaVPN app, navigate to the **Redeem Code** section.
  2. Enter your activation code and click **Redeem**.
  3. The app will activate your service, and your subscription time will be added to your account.

## Step 5: Choose a Server Location

1. In the CicadaVPN app, go to the **Server Location** section.
2. Browse the available server locations and choose the one you want to connect to. You can select the server that provides the best speed or location preference.
3. Click the server name to connect.

## Step 6: Connect to CicadaVPN

1. After selecting a server, click the **Connect** button in the app.
2. The app will establish a secure connection to the VPN server.
3. Once connected, the status will update to **Connected**, and your internet traffic will be securely routed through CicadaVPN.

## Step 7: Customize Settings (Optional)

1. You can adjust the app's settings as needed:
   - **Auto Connect**: Enable this to automatically connect to the VPN when you start your Linux device.
   - **Split Tunneling**: Configure specific apps to bypass the VPN.

## Step 8: Disconnecting from the VPN

1. When you're finished using the VPN, click the **Disconnect** button in the app.
2. The app will disconnect from the server, and your internet traffic will return to its normal state.

## Troubleshooting

- **Connection Issues**: If you are unable to connect, try switching to a different server or ensure your internet connection is stable.
- **Activation Code Problems**: Double-check that you’ve entered the correct code. If issues persist, contact [support@cicadavpn.com](mailto:support@cicadavpn.com) for assistance.

## Step 9: Enjoy Private and Secure Browsing

That’s it! You’re now set up with CicadaVPN on your Linux device. Enjoy safe, private browsing and access to content worldwide.

For any additional questions or support, feel free to contact us at [support@cicadavpn.com](mailto:support@cicadavpn.com).
