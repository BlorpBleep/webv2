# Install Cicada VPN App on Windows

**Last updated:** April 25, 2024

This guide explains how to install and uninstall the Cicada VPN app for Windows users.

## Installation Requirements

- Windows 10 (64-bit) or Windows 11 (64-bit).
- The Cicada app does not work on Windows on ARM.
- For Windows 7 / Windows 8 users, instead of using the Cicada app, you can use the WireGuard app.

## Download and Install the Cicada App

1. Download the latest version of the app from our [official website](https://www.cicadavpn.com/download).
2. Open the downloaded file and follow the prompts to install the app.
3. If you receive a User Account Control warning window like the one below, click on **Yes**.
   ![User Account Control Warning](#)
4. When the installation is completed, click on **Finish** to start Cicada.
   ![Installation Complete](#)

For instructions on how to use our app, continue to read [Using the Cicada VPN App](#).

## Troubleshooting Installation

**Error: "Failed to uninstall a previous version"**

![Uninstall Error](#)

This error can occur if another app holds a lock on the folder `C:\Program Files\Cicada VPN`. To resolve this issue:

- Restart your computer.
- After restarting, try installing the new Cicada version again.

## Uninstall the App (Windows 10/11)

To uninstall the Cicada VPN app from your computer:

1. Click on the **Windows Start** button.
2. Click on the **Settings** icon in the left column and then click on **Apps**.
3. In the list of programs that appears, find **Cicada VPN** and click on it to select it.
4. Click on the **Uninstall** button and confirm that you want to uninstall the app.
5. Windows will proceed to uninstall Cicada and erase its local log files and cache. You'll also have the option to remove your settings.
6. Click on **OK** once it's complete.
